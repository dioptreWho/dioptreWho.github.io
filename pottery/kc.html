Kiln Control

Power Up Display Sequence
  1. Set toggle switch to "ON".
  2. All LED and 7 segments displays will be lighted.
  3. Display input type is shown.
  4. Display range is shown.
  5. Ready for use.

The Home Screen will display at the end of the start-up sequence. This screen can be reached from any programming menu by pressing the A/M key twice. The controller will also revert to the home screen if no key is pressed for 60 seconds.

NOTE: The A/M key should be held for 2 seconds each time it is pressed.

Run Instructions
  1. Follow the Power Up sequence.
  2. Press  SET  repeatedly until ‘Ptn’ displays on the PV screen.
  3. Press   <  .The green digit in the SV display starts to blink.
  4. Press   ∧   or   ∨   to change the pattern number.
  5. Press  SET  to set the number. The SV value will stop flashing.
  6. Press   ∧   to run the program.
